# 📊 图计算系统 Demo - 用户使用手册

欢迎使用 **图计算系统 (Graph Computing System)**。这是一个基于 Web 的轻量级图数据可视化与分析平台，支持复杂网络的可视化展示、图算法分析以及数据的导入导出。

---

## 1. 快速开始

### 🖥️ 界面概览
*   **左侧侧边栏**：操作控制中心。包含搜索、编辑、数据导入导出、算法分析、属性面板。
*   **右侧画布**：图谱展示区域。支持鼠标滚轮缩放、拖拽移动画布、拖拽节点。

---

## 2. 核心功能详解

### 🔍 A. 搜索与定位
在侧边栏最顶部的搜索框中：
1.  输入**节点名称**（支持模糊匹配）或 **节点 ID**。
2.  点击 **🔍 按钮** 或按回车键。
3.  **效果**：系统会自动移动视角聚焦到目标节点，并给予红色发光高亮显示，其余无关节点变暗。
4.  点击 **❌ 按钮**：清除搜索结果，重置画布到默认视图。

### ✏️ B. 图谱编辑 (Editor)
用于手动调整图结构：
*   **+ 节点**：在画布随机位置生成一个新的空白节点。
*   **🔗 连线**：点击进入“连线模式”（按钮变黄）。
    1.  点击第一个节点（起点）。
    2.  点击第二个节点（终点）。
    3.  系统会自动创建一条带箭头的有向边。
    4.  *再次点击按钮可退出该模式。*
*   **🗑️ 删除**：选中画布上的任意节点或连线（选中后边缘变黑），点击此按钮进行删除。

### 💾 C. 数据管理 (Data IO)
支持将当前的图谱保存下来，或从 Excel/CSV 导入数据。

#### 1. JSON 存取 (完整备份)
*   **JSON 导出**：将当前画布上所有节点、关系、位置坐标、颜色样式保存为一个 `.json` 文件。适合**全量备份**。
*   **JSON 导入**：读取备份文件，完美还原上次的工作状态。

#### 2. CSV 导入导出 (Excel 对接)
*   **📥 导 CSV 节点**：上传节点数据表。
    *   *提示：系统会自动询问是“清空当前画布”还是“保留旧数据进行追加”。*
*   **📥 导 CSV 关系**：上传关系数据表。
    *   **智能补全功能**：如果关系表中出现了未定义的节点 ID，系统会自动创建这些节点（显示为黄色背景虚线框），防止连线丢失。

> **📋 CSV 数据格式规范：**
> 建议使用 **UTF-8** 编码的 CSV 文件（可从 Excel 另存为）。
> *   **节点表**：必须包含表头 `id` (唯一编号) 和 `label` (显示名称)。系统支持中文列名识别。
> *   **关系表**：必须包含表头 `source` (起点ID) 和 `target` (终点ID)。

### 🔬 D. 图算法分析 (Analytics)
这是本系统的核心功能，用于挖掘数据背后的规律。

| 图标 | 算法名称 | 功能说明 & 业务含义 | 视觉效果 |
| :--- | :--- | :--- | :--- |
| 🕸️ | **一键整理布局**<br>(ForceAtlas2) | **解开缠绕**：利用物理引力斥力模型，让团在一起的节点自动散开。 | 节点动态运动，最后呈现舒展的结构。 |
| 🔴 | **度中心性**<br>(Degree Centrality) | **寻找热门**：衡量节点的直接连接数。谁连的人最多，谁就是“大V”或枢纽。 | 节点越大、颜色越红，代表连接数越多。 |
| 🌉 | **介数中心性**<br>(Betweenness) | **寻找桥梁**：衡量节点在网络流通中的中介作用。找出连接两个圈子的关键联络人。 | 核心中介节点会显示显著的**紫色双层边框**。 |
| 🎨 | **社区发现**<br>(Louvain) | **划分团伙**：基于关系紧密度自动聚类。 | 不同颜色代表不同的社区/团伙。 |
| ⭐ | **PageRank 排名** | **权威评估**：Google 的核心算法。不仅仅看连接数，更看被谁连接。 | 节点越大，代表权威性越高。 |
| 🛤️ | **最短路径分析**<br>(Shortest Path) | **路径导航**：计算两点之间最少需要几步到达。 | 点击按钮进入模式 → 依次点击起点和终点 → **路径变红高亮**。 |

### 🛠️ E. 辅助工具
*   **重置样式**：保留图数据，但清除所有算法产生的颜色、大小变化，恢复成默认的白色圆点样式。
*   **恢复数据**：**⚠️慎用！** 这会清空你当前的所有操作，并恢复到系统预置的 "Hello World" 演示数据。
